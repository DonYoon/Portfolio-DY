Stock Tracker

Project Description: "Stock Tracker" is an intuitive and user-friendly web application for tracking and analysing stocks in real time. Leveraging a third-party API, the app provides users with an extensive database of stocks, real-time quotes, historical data, and other essential details. It serves as a reliable tool for investors, financial advisors, and individuals to stay updated with the latest changes in the stock market, add stocks to their watchlist, and make informed investment decisions.

Features:

  • Real-Time Stock Tracking: Users can search for stocks using their symbols and get real-time quotes. The stock list updates continuously, reflecting the latest changes in stock prices.
  • Detailed Stock Analysis: For each stock, the app provides comprehensive data, including last traded price, change in price, opening price, high and low of the day, and previous closing price.
  • Interactive Stock Charts: The app includes detailed, interactive charts that represent the stock's performance over 24 hours, 7 days, and 1 year, providing the user with a visual representation of the stock's trends.
  • Watchlist: Users can add stocks to their personal watchlist. This feature enables users to closely track their preferred stocks and quickly access them without searching through the extensive database.
  • Stock Details: For each stock, users can access in-depth information, such as the company's name, country of origin, ticker symbol, exchange it's listed on, industry, IPO date, market capitalization, shares outstanding, and the company's web URL.
  • AutoComplete Search: The AutoComplete search feature provides suggestions based on the user's input, making it easier for the user to find the desired stock.

Key Technologies:

  • React: Used for building the user interface, managing state with hooks, and handling routing with react-router-dom.
  • React-Router-Dom: Used for routing and navigation within the application.
  • React-apexcharts: Implemented for creating interactive and responsive stock charts.
  • axios: Utilised for making HTTP requests to the third-party API - FinnHub.
  • FinnHub API: A stock market API leveraged for real-time and historical stock data, company profiles, and more.
  • Context API: Utilized for managing and sharing state between different components. The WatchListContext provides a centralized way to manage the user's watchlist.
  • react-icons: Used for incorporating intuitive icons within the user interface.
