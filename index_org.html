Inspirational Email Sender

Project Description: This AWS Lambda function sends an inspirational email to the specified recipient using Amazon Simple Email Service (SES). The function is written in Python and leverages the Boto3 library to interact with AWS services.

The Lambda function generates a random inspirational quote from a predefined list and creates an email message with the quote as the body. The email is then sent using Amazon SES as the SMTP server. This function is a great example of using serverless architecture to perform a specific task like sending personalized, automated emails without managing any underlying infrastructure.

Key Technologies:

  • AWS Lambda
  • Amazon Simple Email Service (SES)
  • Python
  • Boto3

    import boto3
    import smtplib
    from email.mime.text import MIMEText
    import random
    
    # Set the SMTP server and sender/receiver email addresses
    SMTP_SERVER = "email-smtp.ap-southeast-2.amazonaws.com"
    SENDER = "don.namkung@gmail.com"
    RECIPIENT = "don.namkung@gmail.com"
    
    peptalk = random.choice(
        [
            "A", "B", "C",
        ]
    )
    
    def lambda_handler(event, context):
        # Create the email message
        message = MIMEText(peptalk)
        message["Subject"] = "You can do it!"
        message["From"] = SENDER
        message["To"] = RECIPIENT
    
        # Send the email using the Amazon SES SMTP server
        client = boto3.client('ses', region_name="ap-southeast-2")
        response = client.send_raw_email(
            Source=SENDER,
            Destinations=[RECIPIENT],
            RawMessage={'Data': message.as_string()}
        )
    
        # Print the response
        print("Email sent! Message ID:", response["MessageId"])

SMS Sender with Textbelt

Project Description: This project utilizes a Python script that sends SMS messages using the Textbelt API. The script is designed to be deployed on AWS Lambda and can be scheduled to send messages periodically. Textbelt is a simple and effective SMS gateway service that enables users to send messages with ease.

The script imports the 'requests' library to make HTTP POST requests to the Textbelt API, while 'schedule' and 'time' libraries allow for scheduling and time management. The user's mobile number and message content are stored in the 'credentials' module to maintain privacy and security.

Key Technologies:

  • Python
  • AWS Lambda
  • Textbelt API
  • requests library
  • schedule library

    import requests
    import schedule
    import time
    from credentials import mobile_number
    
    def lambda_handler(event, context):
        send_message()
    
    def send_message():
        resp = requests.post('https://textbelt.com/text', {
            'phone': mobile_number,
            'message': 'Test message from Lambda.',
            'key': 'textbelt'
        })
        print(resp.json())
