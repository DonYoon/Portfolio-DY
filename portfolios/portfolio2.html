SMS Sender with Textbelt API

Project Description: This project utilises a Python script that sends SMS messages using the Textbelt API. The script is designed to be deployed on AWS Lambda and can be scheduled to send messages periodically. Textbelt is a simple and effective SMS gateway service that enables users to send messages with ease.

The script imports the 'requests' library to make HTTP POST requests to the Textbelt API, while 'schedule' and 'time' libraries allow for scheduling and time management. The user's mobile number and message content are stored in the 'credentials' module to maintain privacy and security.

Key Technologies:

  • Python
  • AWS Lambda
  • Textbelt API
  • requests library
  • schedule library
